A small script with menu without using enter-key 
to control the kenwood ts-590 for easy using
reduced to urgend commands

INSTALL:
- download
- mv cr /usr/local/bin/
- chmod +x /usr/local/bin/cr
...ready to use
  "cr"

looks like:

crig - Helferlein für rigctl/hamlibmögliche Kommandos:
 s - Statusabfrage
 c - COMP - Kompressor ein/aus
 t - Tuner - Tuner ein/aus
 'T - TUNE - tuning
 p - Power - Power high/low
 w - Squelch - open/close(9/10) 
 v - AF LEVEL - 0.05/0.2
 'V - AF - Prozent AF Lautstärke
 a - ANT 1/2
 1 - MEM - Memory 1
 2 - MEM - Memory 2
 3 - MEM - Memory 3
 4 - MEM - Memory 4
 + - MEM - Memory up 
 - - MEM - Memory down
